# econjourney
# 经济学资料分享 & 在线测试平台

## 仓库简介
本仓库专注于西方经济学相关资源汇总与共享，
支持个人上传分享优质学习资源，配套在线题库、章节练习与模拟自测。

## 资源分类
- 微观经济学、宏观经济学
- 课件讲义、复习笔记、课后答案
- 期末试卷、考研真题、模拟试题

## 在线测试板块
- 章节专项练习题
- 经济学基础题库
- 真题套卷模拟测试
- 附带答案与知识点解析

## 参与共建
1. Fork 本仓库
2. 按目录分类上传资料
3. 提交 Pull Request 等待合并
4. 遵守上传规范，拒绝版权违规、低质量灌水内容

## 上传规范
请按对应文件夹归类，命名格式：`课程-年份-资料类型-作者/来源`，保持仓库整洁统一。
